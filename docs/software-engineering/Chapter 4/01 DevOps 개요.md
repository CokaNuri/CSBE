# 01 실현 가능성 분석

## 실현 가능한가?

### 개발 비용

- 인건비
- 하드웨어 비용
- 소프트웨어 도구 비용
- 유지보수 비용 

### 이득

- 비즈니스 프로세스 개선된 영역은 무엇인가?
- 운영의 효율성과 정확성은 어느 정도 증진되는가?
- 비용 절감 효과가 어느 정도 되는가?


### 대안 분석

- 고급 기술자 고용 vs 기존 직원 교육 
- 비용 대 효과 분석 - 실현 가능성 분석

### 기술적 측면
- 기술이나 이론에 대해 제약사항이 없는가?
- 현재 가용한 기술 수준으로 충분한가?


### 스케줄 측면
- 가용한 인력과 자원을 이용해 주어진 기간 내에 완료할 수 있을까?

### 운영적 측면
- 데이터 입력이 가능한가?
- 운영에 대한 역할이나 책임을 수행하기에 관리가자 주저함이 없는가?

### 동기적 측면
- 실제로 사용할 그 사용자가 그 필요성을 인정하고 있는가?

### 법적·윤리적 측면
- 개인 정보 유출과 같은 법적 분쟁의 소지가 발생할 수 있는가?
