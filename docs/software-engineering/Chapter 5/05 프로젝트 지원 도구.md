# 프로젝트 지원 도구

## 프로젝트 관리 기능

> 작업 계획 수립, 팀 구성, 진도 관리 기능 지원

- 계획 기능
- 진도 관리 기능
- 협업 기능
- 보고서 생성 기능

<details><summary>마일스톤이란 무엇인가?</summary>

프로젝트 진행 중 도달해야 하는 진척을 확인할 수 있는 기준점 

</details>

## PMO 도구

- 프로젝트 통합 관리 기능
- 이슈 및 위험 관리 기능
- 성과 관리 기능
- 조직 능력 관리 기능

<details><summary>KPI란 무엇이고 예시는?</summary>

KPI는 목표를 얼마나 달성했는지 측정하는 핵심 지표 <br> 
측정 가능해야 하고, 목표와 직접 연결되어야 함 <br>

MAU, NPS(추천 지수) 등이 있음

</details>

## 엔지니어링 도구

> 분석·설계·코딩·테스트에 사용되는 도구

> UML 기반 객체지향 분석 및 설계 도구, 문서화 도구 등을 포함

| 기능 | 대표 도구 |
|------|-----------|
| 프로젝트 관리  | Jira |
| 협업·의사소통 | Slack |
| PMO  | Planview |
| UML·설계 | StarUML |
| 코딩 |  IntelliJ IDEA |
| 빌드·CI/CD | GitHub Actions |
| 테스트| JUnit|
